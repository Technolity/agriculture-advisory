## Overview

This prompt scaffolds the complete foundation for the "Agricultural Advisory App" — 
an offline-capable mobile app for crop disease detection and agricultural advisory,
targeting smallholder farmers in Kashmir and South Asia.

### Tech Stack
- **Frontend**: React Native + Expo, TypeScript, Redux Toolkit, SQLite
- **Backend**: Node.js + Express, TypeScript, Prisma ORM, PostgreSQL
- **Team**: 3 people (Frontend Dev, Backend Dev, Presenter/DevOps)
- **Timeline**: 48-72 hour hackathon

### Generated Output
- ~87 boilerplate files
- Complete folder structure for both mobile and backend
- Database schema with 8 tables (Prisma)
- 7 API routes with controllers, services, and middleware
- Redux store with 4 slices
- 6 screens, 8 components, 5 hooks, 5 services
- Multilingual support (English, Urdu, Punjabi)
- Offline-first architecture with sync queue
- Edge case handling for 10 categories
- Test scaffolds (structure only, no implementations)
- Documentation (README, Architecture, API, Edge Cases)
